# Project Overview

This skill helps me understand the current project's context by reading a specific file in the project's root directory.

## How it works

When this skill is invoked, I will look for a file named `copilot_overview.md` in the root of the current project.

I will then read this file to get information about:
- The project's purpose
- Key technologies used
- How to run the project
- Important files or directories
- Any other relevant information you have provided.

## When to use

I will automatically try to use this skill at the beginning of our conversations to get up to speed on the project we are working on.
